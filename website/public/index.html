What is PICPIO

PIC development without the pain.

Microchip's PIC microcontrollers are everywhere — but the classic workflow means heavy IDEs and writing to raw registers. PICPIO makes PIC development feel modern and approachable, inside the editor you already use. You get a clean, high-level C API (gpio_write, adc_read, uart1, i2c1…), and PICPIO handles the compiler, programmer, and project setup for you.

How to install

Install it from VS Code.

PICPIO is a VS Code extension. Install it the same way you install any extension — right from the Extensions panel. No downloads to hunt for.

Visual Studio Code — Extensions
🔍 PICPIO
PICPIO ✓
PIC Microcontroller IDE — Build, Flash, Libraries, Simulator
CURIOUS WORM

Open VS Code

Get it free from code.visualstudio.com if you don't already have it.

Open the Extensions panel

Click the Extensions icon in the sidebar, or press Ctrl+Shift+X.

Search “PICPIO”

Type PICPIO in the search box and hit Install on the PICPIO extension by CURIOUS WORM.

Add Microchip's free tools

On first run, PICPIO points you to Microchip's official pages for the XC8 compiler and MPLAB X (for flashing) — free, installed once.

Create a project

Click the PICPIO icon in the sidebar → New Project, pick your chip, and start coding.

Prefer the keyboard? Press Ctrl+P in VS Code and run:

VS Code — Quick Open
ext install picpio.picpio
